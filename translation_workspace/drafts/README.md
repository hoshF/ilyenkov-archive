# drafts

初译草稿目录。

每篇或每本作品按作者和作品分层，例如：

```text
drafts/author-id/work-id/
  translation.json
  draft_zh.md
  notes.md
```

`translation.json` 必须绑定原文路径、版本和 SHA-256；不要复制来源正文。
