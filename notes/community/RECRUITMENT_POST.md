# 招募文章草稿

适用于豆瓣（相关小组、《辩证逻辑》等书的条目页）、知乎（文章或相关问题回答）。发布前按平台调整长度和语气；豆瓣版可以更短，知乎版可保留全文。

## 标题候选

- 有人想一起把伊里因科夫的文本整理清楚吗？

## 正文

苏联哲学家埃瓦尔德·伊里因科夫（Э. В. Ильенков，1924–1979）的中文译介到今天仍然零散：《辩证逻辑》《论偶像与理想》等有旧译本，版次和校订状态不一；大量文本没有中译；"идеальное（观念的/理想的）"这类核心术语在各译本里译法不一，缺一份系统的对照。

我建了一个公开的资料库，目前已经有：

- 《辩证逻辑：历史与理论论文集》《论偶像与理想》的中文整理工程；
- 从 caute.ru 整理的俄文原文 Markdown 语料（含迈丹斯基的研究资料）；
- 已有中文译本的版本线索目录；
- 术语表和翻译风格笔记的初稿；
- 一套来源可追溯、版权可审计的整理流程（公开仓库只发布通过版权审核的内容）。

仓库地址：<https://github.com/hoshF/Ilyenkov-cn>

**找什么人。** 不要求全才，以下任何一项都有用：

- 中文母语、懂俄语：对照原文校订译本，这是最稀缺的；
- 中文母语、不懂俄语：校对中文文本的一致性、考证版本和出版信息、整理术语表；
- 懂英语/德语：对照英译本（如 *Dialectical Logic* 英译）做三方比较；
- 哲学、教育学、心理学背景：维果茨基、活动理论方向的读者会对伊里因科夫的"观念的东西"问题感兴趣；
- 只是读者：阅读感悟和疑问本身就是贡献，很多误译是读者先发现的。

**怎么参与。** 仓库的 Discussions 区有欢迎帖和分类说明，带 `good first issue` 标签的任务不需要任何门槛（有一条甚至不需要俄语：核对来源链接）。长期参与者会受邀加入核心协作组。

**几条原则。** 小步、可追溯、人工校验：不接受大批量未经人工检查的机器翻译；术语定版有讨论流程；所有公开内容经过版权门控。这个项目想做的不是"快速生成一堆译文"，而是给后来者留下一份可以核查、可以继续的工作。

有兴趣的话，从仓库的 README 和欢迎帖开始，或者直接在这条下面留言。
