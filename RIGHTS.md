# Rights and Licensing / 权利与许可

## English

The Ilyenkov Philosophy Text Archive uses layered licensing. The root
[`LICENSE`](LICENSE) is a scope statement, not a blanket license for every
file.

| Material | Default license or policy |
|---|---|
| Project-authored code, scripts, tests, schemas and Git hooks | MIT |
| Project-authored documentation, research notes and original annotations | CC BY-SA 4.0 |
| Project-created factual bibliography, provenance, source, checksum and status metadata | CC0 1.0 |
| Third-party source texts, translations, scans, images and transcriptions | Their own rights status; no project license is implied |

Factual metadata is public by default. Metadata containing substantial
third-party text is controlled content rather than CC0 metadata. Source texts,
translations, scans and media are published only after review of the exact
file, source evidence and SHA-256 checksum.

Project-authored Chinese translations may be offered under CC BY-SA 4.0 only
when the underlying work and the source edition permit that redistribution.
The project cannot license rights it does not own.

“AI-ready” describes technical suitability for auditable research. It does
not grant permission for model training. Training, redistribution and other
uses remain subject to the license or rights statement attached to each
material.

If you are a rightsholder, author, translator, editor or source provider and
believe that a public item should be corrected or removed, open an issue in
the public repository with the affected path and supporting information.

The first reviewed text release covers selected Latin and Dutch Wikisource
transcriptions and DBNL's 1677 *Nagelate schriften*. Redistribution approval
does not mean that these transcriptions are critically edited or fully
collated.

## 中文

伊里因科夫哲学文本库采用分层许可。根目录的 [`LICENSE`](LICENSE) 用于说明适用范围，
并不是覆盖仓库全部文件的统一许可证。

| 材料 | 默认许可或规则 |
|---|---|
| 项目原创代码、脚本、测试、Schema 和 Git hooks | MIT |
| 项目原创文档、研究笔记和原创注释 | CC BY-SA 4.0 |
| 项目整理的事实性书目、来源、校验和与状态元数据 | CC0 1.0 |
| 第三方原典、译文、扫描件、图片和转录文本 | 保留各自权利状态，不自动适用项目许可 |

事实性元数据默认公开。包含较多第三方正文的元数据属于受控内容，不按 CC0 处理。原典、
译文、扫描件和媒体只有在核验具体文件、来源证据和 SHA-256 后才可公开。

项目原创中文译文只有在原作和所依据版本允许公开再分发时，才能以 CC BY-SA 4.0 发布。
项目不能授予自己并不拥有的权利。

“AI-ready”只说明材料适合可审计的计算研究，不代表获得模型训练许可。训练、再分发和其他
使用仍须遵守具体材料的许可证或权利声明。

如你是相关材料的权利人、作者、译者、编辑或来源提供者，认为公开内容需要修正或移除，
请在公开仓库提交 issue，并注明文件路径和可核验依据。

首批完成权利审核的文本包括部分拉丁文、荷兰文 Wikisource 转录，以及 DBNL 收录的
1677 年《Nagelate schriften》。允许公开再分发不代表这些文本已经完成校勘或形成定本。
